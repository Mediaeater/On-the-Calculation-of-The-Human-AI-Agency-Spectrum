## The OKR

**Objective:** Enable tool builders to ship AI features that preserve human agency by default.

**Key Results:**
- Default to Suggest Mode (opt-in) for all generative features
- Track and attribute every AI contribution at the data layer
- Export in standard formats with attribution metadata intact
- Measure agency health (acceptance rates, skill trajectory, mode distribution)
